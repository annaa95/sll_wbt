Overview

This simulation aims to investigate how to learn the dynamical stopping task with a Single-leg robot using provided with a more or less dense set of touch sensors. The simulation is organized into two main controllers, in the Python programming language.

The goal is for the robot to lear a mapping from any arbitrary state to a hopping state with 0 horizontal velocity.

To encourage robust behaviour, noise is injected in sensors' readings, as well as in floor mesh, reinitialized at each episode.

Metrics

t = 00:00:00

The learning metric is log(1+1/|x'|), a function of the apex horizontal velocity. Maximizing the metrics is the goal for this scenario.

The simulation stops after either the robot horizontal speed is less than 0.01 m/s l or if the robot body hit the ground or in any case after 5 consecutive jumps.
